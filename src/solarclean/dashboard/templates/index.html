Notes on dates, weather, and configurations
  • Start and end dates apply to this run only — the saved YAML is never edited. Non-calendar-year periods are useful for experiments, but the comparison deliberately withholds an economic recommendation unless the run covers Jan 1–Dec 31.
  • NASA POWER weather is fetched for the configured site on first use, then cached — only the first run per location needs internet.
  • The default configuration is editable here; the other listed configurations are read-only.
  • Parameter ranges come from the selected configuration's calibration registry — hover an option to see its range.
