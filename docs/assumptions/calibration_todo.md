# Calibration Todo

The following assumptions require later scientific calibration before operational use:

- Riyadh-specific base daily soiling loss rate.
- Seasonal soiling multipliers.
- Heavy dust event probability and loss distribution.
- Rainfall thresholds for partial and strong natural cleaning.
- Rainfall cleaning efficiencies by dust type and storm intensity.
- Bird-dropping event rates by zone and season.
- Mapping from bird-dropping coverage to energy loss.
- Panel module and inverter specifications for the real farm.
- Temperature model coefficients for the installed module/racking type.
- Validation against measured production and local weather station data.

Future calibration should record source data, fitting method, validation split, uncertainty, and the config version produced by the calibration.
